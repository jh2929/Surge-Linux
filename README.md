# Surge 📦 — @jhezdev

![Linux 1K](Linux_1K.jpeg)

La instalación universal de aplicaciones Linux, redefinida. Surge maneja `.deb`, `.rpm`, `.AppImage`, y tarballs en tu espacio de usuario — sin `sudo`, sin contenedores pesados, sin complicaciones.

```bash
curl -fsSL https://raw.githubusercontent.com/jh2929/Surge-Linux/main/install.sh | sh
```

Creado en **6 horas** por un desarrollador de **16 años** con una meta clara: mejorar la experiencia de usuario en Linux eliminando la fricción entre formatos de paquetes y escritorios minimalistas.
