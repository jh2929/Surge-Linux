# Surge 📦 — @jhezdev

![Linux 1K](Linux_1K.jpeg)

Surge revoluciona la experiencia de usuario en Linux. Unifica todos los formatos de instalación bajo una misma interfaz — sin importar qué distribución uses, sin importar qué escritorio tengas. Un solo comando, una sola experiencia, cero complicaciones.

```bash
curl -fsSL https://raw.githubusercontent.com/jh2929/Surge-Linux/main/install.sh | sh
```

Di adiós a los gestores de paquetes fragmentados, a los contenedores pesados, a la dependencia de la terminal para cada instalación. Surge pone el control en tus manos, donde siempre debió estar.

Creado en **6 horas** por un desarrollador de **16 años** con una meta clara: mejorar la experiencia de usuario en Linux. Para todos. En todos los sistemas.
